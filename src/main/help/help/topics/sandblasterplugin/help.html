SandBlaster Plugin

This plugin is based on the SandBlaster tool and requires the SandBlaster tool along with Python2 and Python3 installed with the pip and lief packages.

Prerequisites

  • SandBlaster tool
  • Python2
  • Python3
    • pip package
    • lief package

It is required to specify the paths to the Python2 and Python3 binaries that contain all required packages.

iOS Version Specific Files

Depending on the iOS version, the sandbox operations file source and sandbox profiles file source must be specified as follows:

iOS Version: 4.x, 9.x - 11.x
  • SandBox Operations File Source: kernel sandbox extension
  • SandBox Profiles File Source: kernel sandbox extension (as a bundle)
iOS Version: 5.x - 8.x
  • SandBox Operations File Source: kernel sandbox extension
  • SandBox Profiles File Source: sandboxd
iOS Version: > 12.x
  • SandBox Operations File Source: kernel cache
  • SandBox Profiles File Source: kernel cache (as a bundle)

Logging

There will be a log file for each process of reversing and a log file for each project where SandBlasterPlugin was used, even multiple times.
