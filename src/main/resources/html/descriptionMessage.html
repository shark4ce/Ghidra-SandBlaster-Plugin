Required files for extracting and reversing SandBox Profiles:

  • iOS Version: 4.x, 9.x - 11.x
    - SandBox Operations File Source: kernel sandbox extension
    - SandBox Profiles File Source: kernel sandbox extension (as a bundle)
  • iOS Version: 5.x - 8.x
    - SandBox Operations File Source: kernel sandbox extension
    - SandBox Profiles File Source: sandboxd
  • iOS Version: > 12.x
    - SandBox Operations File Source: kernel cache
    - SandBox Profiles File Source: kernel cache (as a bundle)
